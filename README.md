# Projetos atuais:

1. Sentinela.py
  Criado para um porteiro digitar nome e idade de cada pessoa, com isso ele pode decidir:
  Acima de 18 anos pode entrar, pessoas abaixo disso tem que esperar pela permissão dos pais.

2. pesoEFrete.py
  Esse código converte peso de (Kg) para (g), calcula frete por (Kg), da desconto se a caixa estiver amassada
  e ao final, mostra o total liquido, bruto e o tanto de registros no dia.

3. SGF.py
  Um sistema de análise de feedback com opção de buscar por nome. No terminal ele mostra a maior e menor avaliação,
  quantas pessoas deram a nota máxima e quantos tem o status de 'Pendente', com cores para identificar os números
  de forma mais rápida. Esse foi meu primeiro grande desafio de lógica, aprendi fluxos de repetição e tuplas com ele.

4. InventarioDeSkins.py
  Um sistema de sorteio para coisas de rpg. O usuario pode colocar os nomes dos itens e pessoas nas listas, o próprio projeto irá sortear o objeto escolhido para uma pessoa aleatória. Contem loops para prevenir erros de digitação ou propositalmente digitar errado e opção de recusar o sorteio. Foi uma forma de praticar listas e loops.
